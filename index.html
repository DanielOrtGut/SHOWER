Internet

Internet es una red global de computadoras interconectadas que utilizan protocolos de comunicación estándar para transmitir y recibir datos. Es una red descentralizada y abierta que permite la comunicación y el intercambio de información entre personas, empresas, instituciones y dispositivos en todo el mundo. Internet proporciona una amplia variedad de servicios y recursos, como el correo electrónico, la transferencia de archivos, la navegación web, las redes sociales y mucho más.

PROTOCOLOS

Protocolos usados en Internet

Existen varios protocolos que hacen posible la comunicación en Internet. El más conocido es TCP/IP, pero también se usan otros como FTP, HTTP, HTTPS, SMTP y POP3.

Protocolo de Internet (IP):

Es el protocolo fundamental en Internet. Proporciona direcciones únicas (direcciones IP) a cada dispositivo conectado a la red y establece cómo se envían y se reciben los paquetes de datos.

Protocolo de Transferencia de Hipertexto (HTTP):

Es el protocolo utilizado para la transferencia de datos en la World Wide Web. Define cómo se solicitan y se entregan los recursos, como páginas web, imágenes y archivos. HTTP está basado en el modelo cliente-servidor

Protocolo de Transferencia de Archivos (FTP):

Es el protocolo utilizado para la transferencia de datos en la World Wide Web. Define cómo se solicitan y se entregan los recursos, como páginas web, imágenes y archivos. HTTP está basado en el modelo cliente-servidor

Protocolo de Correo Simple (SMTP):

Es el protocolo utilizado para el envío de correos electrónicos. SMTP se encarga de enviar los correos desde el servidor de correo del remitente al servidor de correo del destinatario.

Protocolo de Acceso a Mensajes (IMAP y POP):

Son protocolos utilizados para acceder a los correos electrónicos desde un servidor. IMAP (Protocolo de Acceso a Mensajes de Internet) permite gestionar los correos electrónicos en el servidor, mientras que POP (Protocolo de Oficina de Correo) descarga los correos al cliente y los elimina del servidor.

Protocolo de resolución de nombres de dominio (DNS):

Es el protocolo encargado de traducir los nombres de dominio (como www.ejemplo.com) en direcciones IP. Permite que los usuarios accedan a sitios web utilizando nombres de dominio en lugar de direcciones IP numéricas.

Conectividad a Internet

Para conectarse a Internet hay varias opciones como la conexión por cable y la inalámbrica. Cada una tiene sus ventajas y desventajas.

Conexión por cable

La conexión por cable es la más rápida y segura, pero puede limitar el movimiento.

Conexión inalambrica

La conexión inalámbrica brinda mayor movilidad pero es menos rápida.

WIFI

¿Qué es el Wi-Fi?

El Wi-Fi se basa en la transmisión de datos a través de ondas de radio de alta frecuencia. Utiliza puntos de acceso inalámbricos, conocidos como routers Wi-Fi, para establecer una red local inalámbrica (WLAN) que permite a los dispositivos compatibles conectarse y comunicarse entre sí.
